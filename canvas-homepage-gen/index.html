Automatic Homepage Generator

Instructions

  1. Construct your link data table in Excel or Google Sheets.
  2. Paste data into Link Data Table form below.
  3. Press Convert button.
  4. HTML source appears in HTML Source Results box below.
  5. Cut-and-paste HTML Source into code editor of webpage.

Link Data

Paste link data into the form below in the same format as this sample link data. The form expects tab-delimited data, the default for copying from Google Sheets. Note that header cells must be included and must match this example.

Here is a description of each column of the link data:

  1. Sequential number of each course session or each week, however you want to organize it. (This could have been auto generated, but this allows you to mess with it.)
  2. Title for the link.
  3. URL or the page (or module) you want to link to. To get this, paste from the browser's Copy Link Address in the context menu.
  4. Date that this topic will be covered. Date is treated as text, so any format is supported.
